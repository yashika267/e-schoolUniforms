== Description ==

Tish4 is a Multi-Purpose Responsive WordPress Theme, a child theme of Tish1 designed to
support Universal Slider Plugin.

== Frequently Asked Questions ==

= How to Install the Theme? =

Manual installation:

1. Upload the theme folder to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for theme name
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Customize to change theme specific options


== Add Slider to Homepage ==

Tish4 theme supports Universal Slider plugin to display Slider. Here is how to create a Slider to your home page:

1. Install and Activate the Slide Anything plugin: https://wordpress.org/plugins/fusion-slider/

2. Navigate to Admin Panel -> Left Menu -> Universal Slider -> Add New

3. Click on 'New Slideshow' and add Image Slide, Title, and Description for each Slide

4. Save Changes

5. Copy shortcode from Right column (section ' How to Use'), i.e. [FSL id=2977]

6. Navigate to Admin Panel -> Left Menu -> Appearance -> Widgets. Insert 'Text' widget into 'Homepage Widget' and insert the shortcode (from step 5.) into it.
